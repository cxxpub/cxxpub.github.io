# 🎁 一份网页礼物

## 给：张明雪

---

### 中文

这是一个网页。
也是一份礼物。

它不需要账号，不需要说明书，
只要被打开，就会开始下雪。

没有特别的功能目的，
只是想在浏览器里，留下一点节日的温度。

---

#### 🎄 这份礼物里有什么？

* ❄️ 会慢慢落下的雪
* 🎄 一棵安静的圣诞树
* 🎶 可以被唤醒的音乐
* 🖼️ 一些被放进页面里的照片
* 🖱️ / 🤏 可以被轻轻触碰的交互

它们不会催促你做任何事，
只是存在着。

---

#### 🧠 关于技术（如果你好奇）

这是一个**纯前端页面**：

* 使用 React 与 TypeScript
* 由 Vite 构建
* 运行在 GitHub Pages 上
* 没有服务器，也没有数据留下

页面被关闭的那一刻，
一切就回到原点。

---

#### 🎁 使用方式

你可以：

* 打开它，看一会儿
* 放着音乐，让它自己播放
* 在某个安静的时间再回来
* 或者只是知道：它存在过

这就够了。

---

#### ✨ 想说的话

这不是一个“项目成果”，
而是一份被包装成网页的心意。

如果你在某个瞬间觉得页面很安静，
那它就已经被正确地打开了。
